############### 2. HELP ########################################################
################################################################################

To use this script type /scripts. When the dialog window has opened click on 
a script in the listbox to view its stats

################################################################################
############### 3. INSTALLATION ################################################
################################################################################

1. Move scriptviewer.mrc your mIRC directory
2. Open mIRC and type /load -rs scriptviewer.mrc
3. type /scripts in mIRC to load the GUI
